vsftpd tester

Fill in your FTP credentials and copy the commands to run in your terminal.

connection settings

1. ping — check connection start here

Just checks if the FTP server responds.

2. list files

Lists files in the remote path. You should see WordPress files.

3. create a test file

Run this first to create a local file to upload.

4. upload file

Uploads the test file to the remote path.

5. verify upload

List files again — your uploaded file should appear.

6. download file

Downloads the file back to your local machine as downloaded.txt.

7. delete test file

Clean up after testing.

8. verify from wordpress container

Check the file actually landed in the WordPress volume.
